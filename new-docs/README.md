# Recycler AI Documentation

Welcome to the Recycler AI documentation repository, optimized for both human users and LLM coding agents. This documentation is structured to provide clear, efficient navigation through our system architecture, extensibility options, and operational guidelines.

## Getting Started

- **Vault Location**: The main documentation vault is located in the `new-docs/` directory.
- **Entry Point**: Begin your journey at `00 - Maps of Content/Recycler AI Overview.md`, which serves as the focal point for navigating through all documentation.

## Documentation Structure

- **Architecture Map of Content (MOC)**: A comprehensive view of the architecture layers and their interactions can be found in `04 - Permanent/Architecture/MOC.md`.
- **Component Guides**: Each key component, such as State Management, Prompt System, Execution Engine, Hybrid Router, and Observability, is detailed in its respective document under `04 - Permanent/Architecture/` and `04 - Permanent/Extensibility/`.
